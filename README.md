# MediQ
Full-Stack Doctor Appointment Booking System (MERN Stack)

Designed a full-stack SaaS-style web application enabling patients to book, view, and manage doctor
appointments, with role-based access for doctors and patients.

Designed normalized database schemas in MongoDB, implemented CRUD operations, and optimized query
handling for appointment scheduling workflows in healthcare application.

Tech Stack: React.js (Vite), Tailwind CSS, Node.js, Express.js, MongoDB, JWT Authentication, Cloudinary
